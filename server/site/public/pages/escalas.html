Escalas

Como principal guia para a música como conhecemos hoje, temos a escala maior natural, porém o que é uma escala e como posso identificar uma?

Uma escala musical é nada mais nada menos do que uma sequência organizada de notas. Você já deve conhecer a escala maior de Dó, ela é composta por:

DÓ - RÉ - MI - FÁ - SOL - LÁ - SÍ - DÓ

Essa escala é amplamente conhecida, porém, existe outra forma de representá-la:

C - D - E - F - G - A - B - C

ainda é a escala maior, porém nomeada apenas com letras onde:

C = DÓ

D = RÉ

E = MI

F = FÁ

G = SOL

A = LÁ

B = SÍ

Como visto na página de Notação, podemos saber a localização das notas por meio das Claves, com essa informação, a escala maior natural de C pode ser representada em uma pauta deste modo:

Onde começamos em C e terminamos em C.

Porém, não temos somente essas notas, existem 12 notas musicais no total, que são:

C - C# - D - D# - E - F - F# - G - G# - A - A# - B - C

Onde '#' significa sustenido.

Podemos também representar essas mesmas notas de forma diferente:

C - Db - D - Eb - E - F - Gb - G - Ab - A - Bb - B

Onde 'b' significa diminuto.

Se você vai usar sustenido ou diminuto, pode depender da sua preferência e escala, mas as duas notas são a mesma, porém representadas de forma diferente:

F# = Gb

Pense como se # é um passo acima, progredindo, já b é um passo para trás, como uma cadência.

Cada avanço de nota (De D para D#, por exemplo) é chamado de semitom, e cada avanço duplo (De D para E, por exemplo) é chamado de tom.

A escala maior natural é composta por 5 tons e 2 semitons, ou seja 8 notas como podemos ver:

C - C# - D - D# - E - F - F# - G - G# - A - A# - B - C

Então podemos dizer que uma escala não é somente composta por notas, mas sim por intervalos entre elas (tom e semitom).

Com isso em mente, a escala maior natural é representada por:

T - T - sT - T - T - T - sT

T = Tom

sT = Semitom

Não existe somente a escala maior de C, mas sim a de todas as 12 notas, como a de E. Se seguirmos o conceito apresentado acima, a escala maior de E é:

E - F# - G# - A - B - C# - D#
